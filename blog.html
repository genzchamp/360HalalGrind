How I Built My First Website Using ChatGPT (Step-by-Step)

By Nasiruddeen Abdulkarim (Oba) — Founder of 360HalalGrind

In the name of Allah, the Most Merciful. I’m Nasiruddeen Abdulkarim, also known as Oba. I created 360HalalGrind.com from scratch — with no money, no coding experience, and nothing but my Android phone, internet access, and ChatGPT as my teacher. Here’s how it all began.

Step 1 — The Vision

Every idea starts with intention. Mine was simple: to build a digital space that represents Halal Grind — Halal wealth, Halal work, and living only with the Permission of The Unique ONE.

I wanted to share my journey, teach what I’m learning, and help others use technology in ways that please Allah. That’s how the name 360 Halal Grind was born — a circle of faith, focus, and financial growth.

Step 2 — Discovering ChatGPT

I had no background in web development. But when I found ChatGPT, I realized I could learn and build at the same time. I started by asking simple questions like:

"ChatGPT, help me create my first website using only my Android phone."

That single prompt opened a new world for me. ChatGPT generated the first HTML and CSS code that became the foundation of my website.

Step 3 — Building With Free Tools

  • GitHub Pages: free hosting to publish my site.
  • Stackblitz: for editing code directly on my phone.
  • Canva: for creating my logo and banner designs.

These tools made it possible to go from zero to live site — completely free.

Step 4 — The First Launch

I copied ChatGPT’s code into a new GitHub repository, pushed my files, and activated GitHub Pages. When I visited my link and saw my site live, I couldn’t stop smiling — Alhamdulillah, it worked!

Step 5 — Expanding the Vision

After the first success, I started building multiple pages (Home, About, Blog, Contact). I wanted bold colors — gold, white, and neon ash on a dark rich background — representing faith and luxury through simplicity.

Every time I got stuck, I returned to ChatGPT, learned something new, and improved the design. Slowly, it started looking professional — something I could truly call mine.

Step 6 — Lessons Learned

  • You don’t need money to start — you need intention and effort.
  • Technology can serve your faith and purpose if you use it right.
  • ChatGPT is more than an AI — it’s a teacher if you learn to ask the right questions.

Final Words

If you’ve ever felt stuck, remember — Allah already gave you the tools and time you need. Use what’s in your hands, stay sincere, and you’ll be amazed how far you can go.
